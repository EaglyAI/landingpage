# Eagly.ai — Fully Designed Investor Deck (PPT Build Sheet)

This document is a **PowerPoint-ready design blueprint** with final slide copy, layout specs, visual system, and presenter notes.

---

## Global Design System

- **Theme:** Dark premium + AI-native
- **Canvas:** 16:9 widescreen (13.333 x 7.5)
- **Primary colors:**
  - Midnight: `#080C18`
  - Surface: `#111827`
  - Indigo Accent: `#6366F1`
  - Text Primary: `#F3F4F6`
  - Text Secondary: `#9CA3AF`
  - Success Accent: `#34D399`
- **Typography:**
  - Headline: Inter / SF Pro Display Bold
  - Body: Inter / SF Pro Text Regular
  - Numbers: Inter SemiBold
- **Type scale:**
  - H1: 54–62
  - H2: 34–42
  - H3: 24–28
  - Body: 20–24
  - Supporting: 14–18
- **Style rules:**
  - 1 core message per slide
  - 3–5 bullets max
  - Large whitespace
  - Use icons + simple shapes instead of dense graphics

## Presenter Notes (Use Across Deck)

- Keep pace at ~35–45 seconds per slide.
- Emphasize **"operator, not software"** in slides 1, 4, 14.
- Use Sara story to anchor emotional context before market/financial slides.
- Keep numbers crisp; avoid over-explaining methodology during first pass.
- End with category-defining ambition, not just feature roadmap.
